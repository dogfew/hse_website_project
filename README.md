# hse_website_project
Установка зависимостей:

`npm i `

Запуск сервера:

`node index` 

Запускается на

http://localhost:3000

При запуске желательно выключать через Ctrl+C, чтобы убить сервер, а не через Ctrl+Z, т.к. тогда порт останется занят и запустить сервер с той конфигурацией, что есть, не получится.
